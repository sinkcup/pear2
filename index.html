sinkcup PEAR2 Channel

Registering the channel: 
pear channel-discover sinkcup.github.io/pear2
Listing available packages: 
pear remote-list -c sinkcup-pear2
Installing a package: 
pear install sinkcup-pear2/package_name
Installing a specific version/stability: 
pear install sinkcup-pear2/package_name-1.0.0
pear install sinkcup-pear2/package_name-beta
Receiving updates via a feed: 
http://sinkcup.github.io/pear2/feed.xml
